Пиццерия
Репозиторий28_pizza_bot
Введение
Сфера услуг старается быть как можно ближе к своим клиентам. Один из лучших инструментов в этой сфере - оформление заказов через чат. Никакой регистрации для новых клиентов, максимально простой интерфейс, быстрый доступ для лояльных клиентов, живое общение. Для малого бизнеса - самое то.

Пиццерия недавно обзавелась таким аккаунтом в Telegram. Клиенты довольны, заказы сыпятся как из рога изобилия, менеджер не успевает отвечать на все запросы. Наступила пора автоматизации.

Один из самых типичных запросов в чате - рассчитать стоимость заказа. Его мы и будем автоматизировать. С помощью бота Telegram выведем в чат меню пиццерии. Прототип интерфейса уже сделан, надо развить успех.

Твоя задача
Переместить каталог пиццерии из кода на Python в реляционную СУБД
Снабдить каталог админкой чтобы менеджеры могли сами добавлять, удалять и редактировать позиции каталога через веб-интерфейс по адресу /admin
Ограничить доступ в админку для любопытных анонимусов
Требования:

Считывание конфигурации приложения из параметров окружения (environment parameters). В частности:
URI базы данных
логин и пароль к админке
Решение поставляется вместе со скриптами решающими следующие задачи:
создание структуры таблиц в базе данных
первичная загрузка данных каталога в базу данных
В публичный доступ размещать бота/сайт пока рано. Достаточно разработческой версии на localhost.

Рекомендации:

SQLite3 - да, это не production ready решение, но на старте разработки вполне сгодится. Благодаря SQLAlchemy миграция на PostgreSQL будет простой
Flask-Admin + Bootstrap для интерфейса админки
HTTP Basic Auth http://flask.pocoo.org/snippets/8/
Руководство по написанию ботов для Telegram
